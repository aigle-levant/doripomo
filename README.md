# doripomo

## Tech used

- Client: React + TypeScript
- Server: Node + Express
- DB: MongoDB + Mongoose
- 3D: [React Three Fiber](https://r3f.docs.pmnd.rs/)
- Auth: Auth0
- Hosting: Vercel + Render
